# AESD Assignments - Moemedi Mokgedi

This repository contains my submissions for the **Advanced Embedded Systems Design** coursework. It demonstrates proficiency in Linux shell scripting, automated testing, and environment configuration.

##  Repository Structure

* **`/finder-app`**: Core logic for the assignment.
    * `finder.sh`: Searches for text strings within a directory tree.
    * `writer.sh`: Creates files with specified content, ensuring path existence.
* **`/conf`**: System configuration files.
    * `username.txt`: Contains the GitHub identity used for automated grading.
